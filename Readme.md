## 🧪 Practicals Overview

### 1. Getting Started with C
Set up the compiler and execute the basic *Hello, World!* program.

### 2. Input/Output Fundamentals
Use `scanf` and `printf` to understand input/output operations and program structure.

### 3. Sum of Two Numbers
Write a program to input two numbers and display their sum.

### 4. Circle Calculations
Calculate the **area** and **circumference** of a circle using user input.

### 5. Arithmetic Operations
Perform addition, subtraction, multiplication, and division on two numbers.

### 6. Expression Evaluation
Evaluate the following expressions:
- V = u + at  
- S = ut + ½ at²  
- T = 2a + √b + 9c  
- H = √(b² + p²)

### 7. Swapping Variables
Swap two variables:
- Using a temporary variable  
- Without using a temporary variable  

### 8. Largest of Three Numbers
Find the largest number using:
- Conditional (ternary) operator  
- If-else statements  

### 9. Switch Case Practice
Use `switch` statements to:
- Check if a character is a vowel or consonant  
- Determine whether a number is positive, negative, or zero  

### 10. Looping Constructs
Use a `while` loop to find the sum of the first *n* natural numbers.
